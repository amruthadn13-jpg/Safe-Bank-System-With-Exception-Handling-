# Safe-Bank-System-With-Exception-Handling-

ay 25 – Safe Bank System (Exception Handling)
🏦 Overview

A Bank Account System built using OOP with added exception handling to prevent crashes and handle invalid user inputs.

🚀 Features
Deposit money safely
Withdraw money with balance check
View current balance
Handles invalid inputs (like letters instead of numbers)
Prevents negative or zero transactions


🧠 Concepts Used
Classes & Objects
try-except (Exception Handling)
Input validation
Conditional statements
Loops (while)


⚙️ How It Works
A BankAccount class is created
Methods:
deposit() → Adds money safely
withdraw() → Deducts money with validation
check_balance() → Displays balance
Exception handling prevents crashes due to wrong input


❗ Error Handling
ValueError → Handles invalid number input
Prevents:
Negative values
Zero transactions
Insufficient balance

💡 Example
Enter amount: abc
Invalid input! Please enter a number.

Enter amount: 500
Amount deposited successfully!
